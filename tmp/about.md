# What is this

I built Conduit, a.NET and C# pipeline, for processing data, following best practice TDD and using three different data sources: RSS, Zotero, and 834. 

Basically, I'm using this to work through ingesting data from any data source. Again, I wrote it as a domain-independent tool that can be used across domains easily, 833 being one of those. That allowed me to explore structured data with strict validation and processing, learning more about 834 and what it is and how it's used.
